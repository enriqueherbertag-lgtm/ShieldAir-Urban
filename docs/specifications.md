# Especificaciones Técnicas - ShieldAir Urban Lung

## Diseño Bio-Térmico Modular

| Parámetro | Valor |
|-----------|-------|
| Altura | 28 m (optimizada para azoteas, mínimo impacto visual) |
| Diámetro | 3.5 m (estructura hexagonal desmontable) |
| Peso total | 4.2 toneladas (6 módulos de 700 kg c/u) |

## Materiales

| Componente | Material | Función |
|------------|----------|---------|
| Estructura | Aleación de aluminio compuesto + grafeno | 30% más liviano que acero, alta resistencia |
| Filtración | Membrana de grafeno + basalto volcánico nanoestructurado | Eficiencia 92% PM2.5 |
| Bioreactor anular | *Scenedesmus dimorphus* + *Chlorella* | Alto rendimiento en condiciones de smog |
| Recuperación de calor | Láminas Peltier + tubos de calor (heat pipes) + intercambiador de placas | Eficiencia térmica optimizada |

## Sistemas integrados

| Sistema | Especificación |
|---------|----------------|
| Ventilación | Ventiladores de baja presión, 12 kW de consumo |
| Control | Automatización con sensores de calidad de aire |
| Monitoreo | Dashboard "ShieldAir Live" en tiempo real |
