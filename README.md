# ShieldAir - Urban Lung System

**Sistema de infraestructura climática para ciudades con emergencia de calidad del aire.**

ShieldAir es una solución de infraestructura verde modular y autofinanciada que transforma edificios estratégicos en “pulmones urbanos activos”, capturando smog a nivel de calle, recuperando calor residual y generando biomasa, sin costo inicial para la municipalidad.

## Resumen ejecutivo

El sistema se instala en azoteas de edificios emblemáticos (hospitales, oficinas gubernamentales, hoteles), integrando directamente el calor recuperado a sus sistemas de calefacción/agua caliente, garantizando un ahorro energético del 40-60% para el edificio ancla.

El municipio solo paga por resultados medibles de mejora de calidad del aire, mientras que los flujos de ingresos por calor, créditos de carbono y biomasa financian la operación y escalabilidad.

**Cronograma total:** 18 meses desde la firma hasta la validación y plan de escalado metropolitano.

## Ciudades objetivo prioritario

- Nueva Delhi, India (PM2.5 anual: ~150 μg/m³)
- Lahore, Pakistán
- Pekín, China
- Ciudad de México, México
- Santiago, Chile
- Jakarta, Indonesia

## Especificaciones clave

| Parámetro | Valor |
|-----------|-------|
| Altura | 28 m |
| Diámetro | 3.5 m |
| Estructura | Aleación de aluminio compuesto + grafeno |
| Filtración | Membrana de grafeno + basalto volcánico nanoestructurado |
| Bioreactor | *Scenedesmus dimorphus* + *Chlorella* |
| Recuperación de calor | Láminas Peltier + tubos de calor (heat pipes) |
| Peso por torre | 4.2 toneladas (6 módulos de 700 kg) |

## Rendimiento por torre

| Parámetro | Valor |
|-----------|-------|
| Aire procesado | 28,000 m³/hora |
| Calor recuperado | 78 kW térmicos netos (ΔT = 6°C) |
| Consumo eléctrico propio | 12 kW |
| Balance neto energético | +66 kW térmicos |
| Biomasa producida | 15 kg/día (≈5.5 toneladas/año) |
| Captura de CO₂ | 1.1 toneladas/día (≈400 toneladas/año) |
| Reducción de PM2.5 en radio 350 m | 40% |

## Modelo financiero

| Concepto | Valor |
|----------|-------|
| Inversión por torre | USD 220,000 |
| Ingresos recurrentes (sin municipio) | USD 980/mes/torre |
| ROI sin pago municipal | ≈6 años |
| ROI con pago municipal | ≈4 años |

### Flujos de ingreso

1. Venta de calor al edificio ancla (50% descuento): USD 600/mes
2. Venta de créditos de carbono (VERRA/Gold Standard): USD 180/mes
3. Venta de biomasa (biofertilizante): USD 200/mes
4. Pago municipal por servicio de aire limpio: USD X/mes

## Hoja de ruta (18 meses)

| Fase | Duración | Actividades clave |
|------|----------|-------------------|
| **0. Preparación estratégica** | Meses 1-3 | MoU con municipalidad, selección de 4 edificios piloto, financiamiento estructurado |
| **1. Fabricación e instalación** | Meses 4-9 | Fabricación modular, instalación "plug & play" en azoteas |
| **2. Operación y validación** | Meses 10-12 | Monitoreo continuo, reporte de impacto, activación de pago por resultados |
| **3. Escalado metropolitano** | Meses 13-18 | Programa municipal de edificios pulmón, expansión a 50 edificios |

## Impacto escalado (50 torres en Nueva Delhi)

| Parámetro | Valor |
|-----------|-------|
| Aire limpio producido | 1.4 millones de m³/hora |
| Calor recuperado | 3.3 MW térmicos (suficiente para 1,000 viviendas) |
| Biomasa generada | 275 toneladas/año (proteína para 2,500 personas/día) |
| Captura de CO₂ | 20,000 toneladas/año |
| Reducción de PM2.5 | hasta 45% |
| Empleos locales generados | 180 |
